# 道州民

道州民⑴

美臣遇明主也

道州民，多侏儒⑵，长者不过三尺余⑶。

市作矮奴年进送⑷，号为道州任土贡⑸。

任土贡，宁若斯？

不闻使人生别离，老翁哭孙母哭儿！

一自阳城来守郡⑹，不进矮奴频诏问⑺。

城云臣按六典书⑻：任土贡有不贡无。

道州水土所生者，只有矮民无矮奴。

吾君感悟玺书下⑼，岁贡矮奴宜悉罢⑽。

道州民，老者幼者何欣欣！

父兄子弟始相保，从此得作良人身⑾。

道州民，民到于今受其赐⑿，欲说使君先下泪⒀。

仍恐儿孙忘使君⒁，生男多以阳为字⒂。
